Descrição do projeto
================================================

Esse projeto foi feito para que os alunos de engenharia do Insper possam treinar exercícios de python
enquanto ainda não instalaram python em seus próprios computadores. 
Exemplos desses exercícios imcluem funções que usam "for", "while", operações matemáticas, listas
e dicionários.


Desenvolvido por Raul Ikeda.
