Guia para alunos
================================================

**Como usar:**

Para entrar no site, use o mesmo usuário e senha que você usa para entrar no blackboard.

Uma vez dentro do site, haverá uma lista de exercícios disponíveis que você pode resolver.
Clique em um deles para ver o que está sendo pedido.

Para responder, crie um arquivo '.py' em seu computador com a função que está sendo pedida, e
envie-o. O site vai então testar a função que você criou e checar se o resultado está como esperado.
